Data Visualisation

I made a chart engine to easily create visualisations for users, which uses the open source ECharts javascript library. They simply have to select their data and choose which chart they want to present it with, given the data is compatible with the chart. The charts are all interactive, being able to remove some elements or hovering over certain ones would change the visualisation to show another story for the data. I was given permission from my internship to use/show this project I made at my time there. All the data on this page are made up, and no way affiliated with any real water data, simply to show what my charts look like.
